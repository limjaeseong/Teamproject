## mini project 주제
+ 마트 장바구니

## mini project 에 대한 소개
+ 마트 상품들을 편리하게 담고 삭제하는 마트 장바구니 기능입니다!

## mini project 가 가지고 있는 대략적인 기능 설명
+ 마트 상품 담기 기능
+ 마트 상품 삭제 기능
+ 장바구니 담긴 상품 조회 기능
+ 장바구니 수정 기능
+ 마트 상품 검색 기능
  + 마트 상품 이름 검색
  + 마트 상품 가격 검색
  + 마트 상품 설명 검색
  + 마트 상품 카테고리 검색
